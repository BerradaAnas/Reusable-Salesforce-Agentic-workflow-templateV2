# Skill: LWC Review

## Purpose

Review Lightning Web Components for UX safety, accessibility, resilience, and secure data handling.

## Review checklist

- Wire and imperative Apex error handling
- Loading states and empty states
- Accessibility labels, roles, and keyboard behavior
- Permission-aware UI behavior
- Client-side input validation and defensive handling
- Data exposure review in UI and network calls
- Safe rendering and state management
- Apex integration boundaries and failure handling
- Jest test readiness

## Output

- Findings by severity
- File references
- Required fixes
- Missing tests
- Manual validation notes

## Guardrails

- Do not modify files unless ACT MODE implementation is approved.
- Do not ignore accessibility or security findings.
