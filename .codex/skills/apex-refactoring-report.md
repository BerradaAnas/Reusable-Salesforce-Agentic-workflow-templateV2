# Skill: Apex Refactoring Report

## Purpose

Produce consistent analysis, refactoring, test, and validation summaries for Apex cleanup work.

## Required sections

- Target class
- Analyzer report link
- Refactoring goal
- Public interface preserved
- Files changed
- Safe refactors applied
- Tests updated
- Commands to run
- Risks
- Rollback notes
- Next steps

## Reporting expectations

- Link evidence to the analyzer report when it exists.
- Distinguish approved scope from follow-up work.
- Call out residual risks and no-go items clearly.
- Keep rollback notes actionable and concise.
